🌐 CSS 선택자 & 레이아웃 핵심 정리

1. CSS 적용 방식

인라인 스타일: <span style="color:red; font-weight:bold;">이 텍스트는 빨간색 인라인 스타일</span>

이 텍스트는 빨간색 인라인 스타일

  • 인라인 스타일: 태그 내 style=""로 직접 지정
  • 내부 스타일: <style> 태그 안에 CSS 작성
  • 외부 스타일: 별도의 .css 파일 연결

3. 조합 선택자

div[class^="user"]: 시작값 'user'
div[class$="info"]: 끝값 'info'
div[class~="pro"]: 단어 포함 'pro'

div[class^="user"] { background-color: lightgray; }
div[class$="info"] { color: red; }
div[class~="pro"] { background-color: coral; }
  • ^=: 시작값
  • $=: 끝값
  • ~=: 단어 포함

4. 상태 기반 선택자


input:focus { background-color: yellow; }
input:hover { background-color: green; }
  • :focus: 입력 포커스 시
  • :hover: 마우스 오버 시

5. 구조 선택자

  • li:nth-child(2): 두 번째 자식 선택
  • p:only-of-type: 해당 태그가 유일할 때
  • 예시:
    • 첫째 항목
    • 둘째 항목 (bold & blue)
    • 셋째 항목

이 문장은 유일한 <p> 태그이므로 색상이 달라요.


li:nth-child(2) { font-weight: bold; }
p:only-of-type { color: blue; }
