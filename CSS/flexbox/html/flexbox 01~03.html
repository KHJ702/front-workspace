Flex 아이템 전용 속성 핵심 정리

1. order: 아이템 순서 지정 🔀

순서가 달라져도 글자 색과 위치로 구분 가능

order: -1 (앞으로 당김)
order: 0 (기본 순서)
order: 1 (뒤로 밀기)

2. flex-grow: 아이템 팽창 비율 ↔️

빈 공간 채우는 비율에 따라 아이템 넓이 차이

grow: 1
grow: 2
grow: 3

3. flex-shrink: 아이템 수축 비율 ↘️

컨테이너 폭이 줄어들 때 수축 속도 다름

shrink: 0 (수축 안함)
shrink: 1
shrink: 2
shrink: 3

4. flex-basis: 아이템 기본 크기 📐

기본 크기(px, %)

basis: 150px
basis: 10%
basis: 25%

5. flex: 축약 속성 ⚡

수축X, 팽창X, 기본크기 100px 설정

flex: 0 0 100px
grow: 1

6. align-self: 개별 아이템 교차축 정렬 ↕️

각 아이템의 교차축 위치 조절

self-start (위)
self-center (중앙)
self-end (아래)
